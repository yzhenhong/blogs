## Nuxt.js 基础

### 什么是 Nuxt.js
Nuxt.js 是一个基于 Vue.js 的通用应用框架，它提供了许多开箱即用的功能，帮助开发者快速构建现代化的 Web 应用。

### 主要特性
1. 服务端渲染 (SSR)
2. 静态站点生成 (SSG)
3. 自动路由
4. 中间件支持
5. 模块化架构

### 版本对比
1. Nuxt 2
   - 稳定版本
   - 完整的生态系统
   - 丰富的插件支持

2. Nuxt 3
   - 基于 Vue 3
   - 性能提升
   - 更好的 TypeScript 支持
   - 新的组合式 API
